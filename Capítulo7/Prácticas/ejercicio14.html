Pseudocódigo

Solicitar los datos del empleado.
Convertir de cadena a entero.
Si el número de horas es mayor a 40.
Multiplicar 40 por la cuota ingresada.
Restar 40 al número total, el resultado multiplicarlo por la cuota ingresado.
Sumar ambas multiplicaciones.
Mostrar el resultado.
Caso contrario.
Multiplicar las horas por la cuota.
Mostrar el resultado.

Código JavaScript

var hora, hr, hrex, cuota, sal, sueldo;
hora = window.prompt("Ingrese las horas trabajadas del empleado:","0");
hr = parseInt(hora);
cuota = window.prompt("Ingrese la cuota normal del empleado:","0");
sal = parseInt(cuota);
if(hr > 40){
hrex = hr - 40;
sueldo = 40 * sal;
sueldo += (hrex * (sal * 1.5));
}else{
sueldo = hr * sal;
}
document.writeln("< h3 >El sueldo bruto del empleado es de: " + sueldo + "< /h3 >");
