Ejercicio 3.3 de autoevaluación

Complete las siguientes oraciones:

  1. El atributo ________ (que se utiliza en un solo elemento input en un formulario) resalta de manera automática el elemento input y, si se usa de manera apropiada, coloca el cursor en el campo de texto después de que el navegador carga y despliega la página.


    2. Respuesta: autofocus.



  2. Los nuevos tipos input de HTML5 tienen ________ del lado del cliente.


    3. Respuesta: validación automática.



  3. El tipo input ________ permite al usuario introducir un valor numérico.


    4. Respuesta: number.



  4. El tipo input ________ se valida automáticamente de manera intrínseca cuando lo despliega el navegador como un control deslizante, ya que el usuario no puede mover el control fuera de los límites del valor mínimo o máximo.


    5. Respuesta: range.



  5. El atributo ________ puede usarse en los tipos input para completar de manera automática la información del usuario con base en datos de entrada anteriores.


    6. Respuesta: datalist.



  6. El elemento ________ proporciona una leyenda para la imagen en el elemento figure.


    7. Respuesta: figcaption.



  7. El elemento summary muestra una flecha que apunta a la derecha enseguida de un resumen o leyenda cuando el documento se despliega en un navegador. Al hacer clic en la flecha ésta apunta hacia abajo y revela el contenido del elemento ________.


    8. Respuesta: details.



  8. El elemento mark nos permite ________.


    9. Respuesta: resaltar texto.
