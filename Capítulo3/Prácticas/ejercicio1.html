Ejercicio 3.1 de autoevaluación

Complete las siguientes oraciones:

  1. El tipo input color permite al usuario introducir color. Al momento de escribir este libro, la mayoría de los navegadores despliegan el tipo input color como un campo de texto en donde el usuario puede introducir un ________.


    2. Respuesta: código hexadecimal.



  2. El atributo ________ nos permite colocar texto temporal en un campo de texto.


    3. Respuesta: placeholder.



  3. Si desea anular la validación, puede agregar el atributo formovalidate al tipo input ________.


    4. Respuesta: submit.



  4. El atributo ________ obliga al usuario a introducir valores antes de enviar el formulario.


    5. Respuesta: required.



  5. Por lo general se muestra el control ________ para el tipo input number e incluye sólo los números válidos.


    6. Respuesta: spinner.



  6. El tipo input ________ permite al usuario introducir una hora, minuto, segundo y fracción de segundo.


    7. Respuesta: time.



  7. El elemento ________ proporciona opciones de entrada para un elemento input text.


    8. Respuesta: datalist.



  8. El elemento ________ describe contenido que está separado del contenido principal de la página y que posiblemente podría utilizarse o distribuirse en otra parte, como un artículo de noticias, mensaje de foro o entrada de blog.


    9. Respuesta: article.



  9. El elemento ________ describe el texto que por lo general aparece en la parte inferior del contenido o de un elemento section.


    10. Respuesta: footer.



  10. El elemento ________ indica el lugar apropiado para dividir una palabra cuando el texto se ajusta en varias lineas.


    11. Respuesta: wbr.
