Ejercicio 3.2 de autoevaluación

Indique si cada uno de los siguientes enunciados es Verdadero o Falso. Si es Falso, explique por qué.

  1. Cualquier tipo input particular de formulario de HTML5 debe desplegarse de manera idéntica en todos los navegadores compatibles con HTML5.


    2. Respuesta: Falso, esto depende de cada navegador.



  2. Cuando se coloca el enfoque en el campo de texto (es decir, si el cursor está en el campo de texto), se envía el texto del atributo placeholder al servidor.


    3. Respuesta: Falso, no se envía, solamente que el usuario escriba el mismo texto.



  3. No es necesario incluir el atributo autofocus en sus formularios.


    4. Respuesta: Verdadero.



  4. Los nuevos tipos input de HTML5 tienen validación automática del lado del cliente, con lo cual se elimina la necesidad de agregar secuencias de comandos (script) complicadas a sus formularios para validar la entrada del usuario y se reduce la cantidad de datos inválidos que se envían.


    5. Respuesta: Verdadero.



  5. El tipo input range se valida automáticamente de manera intrínseca cuando el navegador lo despliega como un control deslizante, ya que el usuario no puede mover el control fuera de los límites del valor mínimo o máximo.


    6. Respuesta: Verdadero.



  6. HTML5 cuenta con validación automática para el tipo input tel.


    7. Respuesta: Falso, se debe especificar el formato del número telefónico a validar.



  7. Si el usuario introduce en URL con un formato enapropiado en un tipo input url, no se validará. HTML5 no valida si el URL introducido existe en realidad.


    8. Respuesta: Verdadero.



  8. El elemento nav muestra un menú desplegable de hipervínculos.


    9. Respuesta: Falso, agrupa vínculos de navegación.



  9. El elemento header puede usarse sólo una vez en una página.


    10. Respuesta: Falso, se puede ocupar varias veces en una sóla página.



  10. Los elementos nav pueden anidarse en un elemento aside.


    11. Respuesta: Verdadero.



  11. Podemos usar el elemento brk puede evitar dividir palabras en lugares incorrectos.


    12. Respuesta: Falso, el elemento que se ocupa para esa función es wbr.
