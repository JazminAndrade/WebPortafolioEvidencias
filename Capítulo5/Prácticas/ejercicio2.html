Ejercicio 5.2 de autoevaluación

Indique si cada uno de los siguientes enunciados es verdadero o falso. Si es falso, explique por qué.

  1. La regla @font-face especifica que se utilizará una fuente incrustada cuando el elemento se despliegue en una pantalla de computadora.


    2. Respuesta: Falso. La regla @media screen especifica que se utilizará una fuente incrustada cuando el documento se despliegue en una pantalla de computadora.



  2. Podemos usar gradientes en cualquier propiedad que acepte una imagen.


    3. Respuesta: Verdadero.



  3. Un gradiente horizontal cambia en forma gradual de arriba hacia abajo.


    4. Respuesta: Falso. Un gradiente horizontal cambia en forma gradual de izquierda a derecha.



  4. Podemos agregar líneas entre columnas mediante el uso de la propiedad column-gap.


    5. Respuesta: Falso. Podemos agregar líneas entre columnas mediante el uso de la propiedad column-rule.



  5. La regla @media determina el tipo y tamaño del dispositivo en el que se despliega la página. Cuando el navegador examina la regla, el resultado es verdadero o falso. Los estilos de la regla se aplican sólo si el resultado es falso.


    6. Respuesta: Falso, los estilos de la regla @media se aplican sólo si el resultado es verdadero.



  6. Para agregar múltiples imágenes de fondo a un elemento, usamos la propiedad background-position para especificar en dónde colocar cada imagen mediante el uso del modelo de cajas.


    7. Respuesta: Falso. La propiedad back-ground-origin especifica en dónde se va a colocar cada imagen mediante el uso del modelo de cajas.
