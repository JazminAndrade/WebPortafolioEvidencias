Ejercicio 2.2 de autoevaluación

Complete las siguientes oraciones:


  1. El elemento ________ inserta una regla horizontal.


    2. Respuesta: hr



  2. Para marcar un superíndice se utiliza el elemento ________ y para marcar un subíndice se utiliza el elemento ________.


    3. Primer respuesta: sup .
    Segunda respuesta: sub



  3. El elemento de encabezado menos importante es ________ y el elemento de encabezado más importante es ________.


    4. Primer respuesta: h6.
    Segunda respuesta: h1



  4. El elemento ________ marca una lista desordenada.


    5. Respuesta: ul



  5. El elemento ________ marca un párrafo.


    6. Respuesta: p



  6. El atributo ________ es un elemento input inserta un botón que, cuando se presiona, restablece el contenido del formulario.


    7. Respuesta: reset



  7. El elemento ________ marca una fila de una tabla.


    8. Respuesta: tr



  8. Los ________ se usan por lo general como caracteres de enmascarado en un cuadro de contraseña.


    9. Respuesta: Los asteriscos.
