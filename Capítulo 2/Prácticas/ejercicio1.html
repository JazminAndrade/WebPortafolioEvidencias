Ejercicio 2.1 de autoevaluación

Indique si cada uno de los siguientes enunciados es Verdadero o Falso. Si es Falso, explique por qué.


  1. Una lista ordenada no puede anidarse dentro de una lista desordenada.


    2. Falso, pueden anidarse mutuamente sin ningún problema, ninguna impide el correcto trabajo de la otra.



  2. El elemento br representa un salto de línea.


    3. Verdadero.



  3. Los hipervínculos se denotan mediante elementos link.


    4. Falso, los hipervínculos se denotan mediante elementos a.



  4. La anchura de todas las celdas de datos en una tabla debe ser igual.


    5. Falso, el tamaño de las celdas puede variar dependiendo la anchura que se le asigne a la columna.



  5. Estamos limitados a un máximo de cinco vínculos internos por página.


    6. Falso, no hay un limite en la cantidad de hipervínculos por página.
